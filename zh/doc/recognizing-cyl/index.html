              # 认识Cylheim

              Tip：为了方便，在以下内容中部分Cylheim会用缩写Cyl替代。

              # 开始之前

              Cylheim仅支持Windows，因此请尽量确保自己电脑运行推荐Windows10以上的系统。低于Windows10的系统运行Cylheim无法确保稳定。

              # 安装Cylheim

              Cyl可以通过以下两种方式安装：

              • 微软商店（Microsoft Store）open in new window
              • Github Releaseopen in new window

              # 通过微软商店安装

              打开Microsoft Store（可以在开始菜单中找到），搜索Cylheim并点击 安装 按钮，MS Store会自动完成剩余安装步骤。

              # 通过GitHub Release下载安装

              如果觉得MS Store的下载速度太慢（而且经常出一些莫名其妙的错误——拜GFW所赐），不想把软件安装到C盘，或者因为运行的是LTSB/LTSC系统没有MS Store，那么可以通过GitHub下载安装。

              1. 打开上面 提供的GitHub链接，跳转到Release页面
              2. 下拉找到Assets，在两个.exe安装包里选择适用于自己系统架构（32/64位）的版本，点击链接下载

              关于如何查看系统架构，以及如何使用ghproxy等工具加速下载，这里就不加赘述了

              1. 下载完成后，双击程序运行，根据提示一步步操作即可

              Cyl需要.Netopen in new window 6.0桌面运行时环境，若没有提前安装，则安装程序会在确认后自动下载安装（很智能吧）

              现在Cylheim应该已经安装到了电脑上。
